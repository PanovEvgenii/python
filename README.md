# Exclusive Content Stacy Landing

## Что сделано
- Тёмный минималистичный одностраничник с премиум-стилем.
- 2 большие кнопки: `18+` и `Telegram`.
- Модалка подтверждения возраста (18+).
- После подтверждения: блюр пропадает, текст меняется на «Эксклюзивный контент», открывается Fanvue.
- Золотистые искорки + плавные анимации.
- Нижняя анимированная зона: 3 случайные фразы за цикл (fade in/out).
- Автоопределение языка устройства (`ru`/`en`).
- **Общий глобальный счётчик** на базе CountAPI.

## Структура проекта
- `index.html` — структура страницы
- `styles.css` — дизайн
- `script.js` — логика, мультиязык, анимации, счётчик
- `assets/` — фото

## Фото (обязательно)
Положите в `assets/` файлы с точными именами:
- `background.jpg` — фон
- `adult.jpg` — фото на кнопке 18+
- `telegram.jpg` — фото на кнопке Telegram

## Как посмотреть готовый вариант локально
```bash
python3 -m http.server 8080
```
Откройте в браузере:
- `http://localhost:8080`

## Пошагово: как довести проект до конкретной ссылки (GitHub Pages)

### Шаг 1. Создайте репозиторий
1. На GitHub нажмите **New repository**.
2. Название репозитория задайте сразу красивое, например: `stacy-vip`.
3. Сделайте репозиторий **Public**.
4. Нажмите **Create repository**.

### Шаг 2. Загрузите проект в репозиторий
В папке проекта выполните:
```bash
git init
git add .
git commit -m "Initial landing"
git branch -M main
git remote add origin https://github.com/<YOUR_USERNAME>/stacy-vip.git
git push -u origin main
```

### Шаг 3. Включите GitHub Pages
1. Откройте репозиторий → **Settings** → **Pages**.
2. В разделе **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
3. Нажмите **Save**.

### Шаг 4. Получите ссылку
Через 1–3 минуты появится ссылка:
- `https://<YOUR_USERNAME>.github.io/stacy-vip/`

Это и есть ваша рабочая ссылка для Instagram/Threads/TikTok/Facebook.

## Как сделать ещё более «красивую» ссылку
Если нужна ссылка без `github.io`:
1. Купите домен (например `stacyvip.me`).
2. В **Settings → Pages** добавьте **Custom domain**.
3. Пропишите DNS у регистратора по инструкции GitHub.
4. Включите **Enforce HTTPS**.

Итог: ссылка будет вида `https://stacyvip.me`.

## Как работает общий счётчик
- Счётчик общий для всех пользователей (не локальный).
- Используется CountAPI: инкремент при каждом открытии страницы.
- Ключ счётчика содержит текущий месяц (`YYYY-MM`), поэтому в новом месяце счёт начинается заново с 0 автоматически.
- Месяц берётся по UTC.
