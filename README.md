# Exclusive Content Stacy Landing

## Что уже сделано
- Тёмный минималистичный одностраничник.
- 2 крупные кнопки (18+ и Telegram).
- Модальное окно 18+ с подтверждением возраста.
- До подтверждения: блюр на 18+ карточке, после подтверждения: блюр снимается и меняется подпись.
- Золотые «искорки» + плавные анимации.
- Бегущие мотивационные фразы: 3 случайные фразы за цикл.
- Автоопределение языка (RU/EN).
- Счётчик посетителей за месяц (в localStorage браузера), сбрасывается каждый месяц.

## Структура
- `index.html` — разметка
- `styles.css` — стили
- `script.js` — логика
- `assets/` — изображения

## Как запустить локально
1. Откройте папку проекта.
2. Дважды кликните `index.html`.

или через Python:
```bash
python3 -m http.server 8080
```
Затем откройте `http://localhost:8080`.

## Публикация на GitHub Pages (красивая ссылка)
1. Создайте новый репозиторий на GitHub, например `stacy-links`.
2. Загрузите файлы проекта в корень репозитория.
3. Перейдите: **Settings → Pages**.
4. В блоке **Build and deployment** выберите:
   - Source: `Deploy from a branch`
   - Branch: `main` и папка `/ (root)`
5. Нажмите Save и подождите 1–2 минуты.
6. Ссылка будет вида:
   - `https://<your-username>.github.io/stacy-links/`

### Чтобы ссылка была ещё красивее
- Используйте короткое имя репозитория, например `vip`:
  - `https://<your-username>.github.io/vip/`

### Максимально красивая ссылка через свой домен
1. Купите домен (например, `stacy-vip.com`).
2. В **Settings → Pages** добавьте Custom domain.
3. У регистратора домена настройте DNS (A/CNAME) по инструкции GitHub.
4. Включите **Enforce HTTPS**.

## Важно про счётчик
Счётчик сейчас считает **локально на устройстве пользователя** (localStorage). Это быстро и бесплатно, но:
- у каждого пользователя своё число;
- при очистке браузера сбрасывается.

Если нужен **общий реальный счётчик для всех**, нужен бэкенд (например Supabase/Firebase/Cloudflare Worker).
